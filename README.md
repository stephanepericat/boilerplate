# Boilerplate

A webpack boilerplate

## Installation

```shell
npx degit github:stephanepericat/boilerplate ./my-project
cd ./my-project
npm i
```

## Tasks

### Build

```shell
npm run build
```

### Dev

```shell
npm run dev
```

### Test

```shell
npm test
```

## Features

- Webpack dev server
- Jest unit tests
- Sass support
- Travis CI integration
- Coveralls integration
